    关于架构

    Minecraft Windows 版仅支持Windows 10及以上的系统，且每个安装包均仅支持一个架构。比如 Minecraft 1.19.0.5 包括三种安装包，分别支持 x64, x86, arm 三种架构。请根据自己的设备情况做出选择。

    如何在Windows系统中查看设备架构
    1. 按下键，或点击任务栏的图标，或点击任务栏的搜索框。
    2. 输入“系统信息”，并选择“系统信息”打开。
    3. 在弹出的窗口中找到“系统类型”项目，该字段就是当前系统的架构。如“基于x64的电脑”表示你的设备支持x64。
